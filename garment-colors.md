# TRESR Garment Colors Reference

This document serves as the definitive source of truth for all garment colors available on TRESR.com and their corresponding Cloudinary image mappings.

## Color Standardization Key

Our standardized color names map to various source names:
- **Black** = Black, Deep Black
- **White** = White  
- **Navy** = Navy, Classic Navy, Midnight Navy
- **Natural** = Natural, Bone
- **Dark Grey** = Dark Heather Gray, Charcoal, Heather Grey
- **Light Grey** = Gray Heather, Grey, Gusty Grey
- **Cardinal Red** = Red, Cardinal Red
- **Gold** = Gold, Antique Gold
- **Alpine Green** = Alpine Green, Deep Forest Green
- **Army Heather** = Army Heather
- **Royal Blue** = Royal Blue, True Royal, Blue
- **Pink** = Pink, Desert Pink
- **Cotton Candy** = Cotton Candy
- **Mint** = Mint, Sage
- **Black Camo** = Black Camo
